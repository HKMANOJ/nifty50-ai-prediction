Expected filenames:

1. `nifty50_history.csv`
   Minimum columns: `date,open,high,low,close`

2. `india_vix.csv`
   Minimum columns: `date,close`

3. `fii_dii.csv`
   Minimum columns: `date,fii_net_crore,dii_net_crore`

4. `gift_nifty.csv`
   Minimum columns: `date,last,change_pct`

5. `usdinr.csv`
   Minimum columns: `date,reference_rate`

6. `world_signals.json`
   Required bundle for world-news and global-risk context.

Auto-generated by the downloader for the live chart:

7. `nifty50_intraday.json`
   Real `1m` and `5m` NIFTY candle data used by the live intraday chart and next-candle action card.

8. `nifty50_components_live.json`
   Real NIFTY 50 component-stock data used by the heavyweight watchlist plus the `Top 10 Bullish` and `Top 10 Bearish` dashboard lists.

Example `world_signals.json`:

```json
{
  "us_markets": { "change_pct": 0.42 },
  "brent": { "change_pct": -0.58 },
  "news_sentiment": {
    "india": 0.36,
    "global": 0.08
  },
  "domestic_factors": [
    {
      "factor": "RBI commentary steady",
      "impact": "neutral",
      "detail": "No fresh hawkish shock in the latest official commentary."
    }
  ],
  "geopolitical_factors": [
    {
      "factor": "Oil shock risk contained",
      "impact": "positive",
      "detail": "No immediate escalation in the current world-signals file."
    }
  ]
}
```

Notes:

- The downloader now fetches the required files from live sources and writes them here in collector-ready format.
- `nifty50_intraday.json` is optional for the collector, but it is required if you want the full live chart experience in the UI.
- `nifty50_components_live.json` is optional for the collector, but it is required if you want the full component-stock dashboard in the UI.
- This collector is intentionally file-based so the UI stays simple and auditable after the data has been fetched.
- The collector is now strict: if any required file is missing or invalid, it exits with an error and does not create a snapshot.
- You can generate a real `world_signals.json` with:

```bash
python3 /Users/hkmanoj/Documents/Codex/2026-05-29/files-mentioned-by-the-user-nifty50/build_world_signals.py
```

- `NEWSAPI_KEY` is optional for the world-signal builder.
- `FRED_API_KEY` is optional and only used as a backup if Yahoo Finance is unavailable for the US/Brent series.
